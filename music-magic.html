Music Magic Plugin.

Project description

Music Magic is an AI-powered VST3 plugin that integrates generative music directly into digital audio workstations such as Ableton Live.

The plugin is built with JUCE, an audio application framework written in C++. When the user generates new music, the application executes a python script which generates music using a tailored version of Riffusion, a model optimised for generating a spectrogram which is an image representation of audio.

This allows producers to generate, replace, extend and in-fill audio using text prompts and mode-specific controls. The plugin provides an intuitive workflow for previewing, trimming and importing AI-generated audio without leaving the music production environment.
