Surveillance App.

Project description

This project involved building a low-cost distributed system for tracking people across multiple Raspberry Pi cameras.

It combines YOLOv8-nano, DeepSORT and Kalman filtering to perform real-time detection and maintain identities across camera views without transmitting raw video centrally.

My contribution focused on designing and developing the Flutter mobile application that supports multi-camera RTSP streaming, live statistics via WebSockets, system configuration and a prototype dangerous-person detection feature.
